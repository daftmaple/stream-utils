# Stream Utils

## Chatbox

Usage:

```
<BASE_URL>/chatbox/<channel>?align=[left|right]&slideOutDelay=<seconds>
```

Query Params

| Key           | Value Type | Valid value       |
| ------------- | ---------- | ----------------- |
| align         | string     | `left` or `right` |
| slideOutDelay | number     |                   |
